游戏内菜单¶

在视觉小说中，玩家经常会遇到各种各样决定游戏展开的选项。Ren'Py语言中有一个menu命令，可以让您轻松地制作这些选项。

以下是menu命令的例子：

menu:
     "我该怎么办呢？"

     "喝咖啡":
         "我喝了些咖啡，直到最后一滴都意犹未尽。"

     "喝茶":
         $ drank_tea = True

         "我喝了些茶，不知为何想起了些不好的事情。"

     "下跪":
         jump genuflect_ending

label after_menu:

     "我的早晨开始了。"

menu命令由menu关键字开始。后面可以跟一个标签名称，这等价于在menu命令前使用一个label命令。如：

menu drink_menu:
    ...

menu命令后面应该带一个区块。这个区块可以包含say命令，且必须带至少一个选择肢。如果使用了say命令，该行文本将与选择肢同时显示在屏幕上。

选择肢 选择肢指的是玩家在某个选项中所能够做出的某一个具体选择。选择肢由一个字符串开始，可以带一个if从句来决定它是否出现。选择肢由一个冒号结束，然后后面带一个区块，内容是该选择肢被选中后执行的Ren'Py命令。

当脚本执行到该区块的结束部分时，脚本执行器将会从menu命令的后面继续执行下去。

if从句由关键词if以及一个python表达式组成。只有该表达式的值为True时，该选择肢才会出现。在下面这个选项中：

menu:
    "向左":
        ...
    "向右":
        ...
    "起飞" if drank_tea:
        ...

第三个选择肢仅在drank_tea这个变量为True的时候出现。
