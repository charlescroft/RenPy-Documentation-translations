条件语句¶

Ren'Py中有许多可以根据表达式的值来控制脚本运行的命令。我们已经见过一些不需要条件就能够控制脚本运行的命令了，如jump, call和return命令。

需要注意的是，这一节我们讨论的是Ren'Py中使用的命令，而不是python代码中使用的命令。在Python代码中，您不能使用Ren'Py里的这些命令（虽然看起来十分相似）。

If命令¶

if命令的作用是，若提供python表达式为真，则执行之后区块内的代码。它由一个if从句，若干个elif 从句，以及在最后的可选的else从句组成。

每个从句都应该在自己的逻辑行内，后面跟着一个区块。if和elif从句后面接一个表达式。所有的从句均以冒号结束。

下面是一些栗子:

if flag:
    e "死亡flag已经立起来了哦！"

if points >= 10:
    jump best_ending
elif points >= 5:
    jump good_ending
elif points >= 1
    jump bad_ending
else:
    jump worst_ending

在if命令中的表达式的求值顺序是从上向下的。当有一个表达式的值为True时，该从句对应的区块将被执行。当脚本运行到该区块的结束部分时，Ren'Py将运行在if命令之后的脚本。.

如果所有的表达式的值均为False，else 从句对应的区块将被执行（如果有else从句的话）。

While命令¶

该命令会在表达式的值为True时反复运行一个区块的代码，直到表达式的值不再为True为止。比如：

$ count = 10

while count > 0:

    "T-minus [count]."

    $ count -= 1

"Liftoff!"

while True:

    "此乃永无终焉之歌。"
    "谛听吧，吾之半身哟！"

该表达式会在脚本第一次运行到这里以及每次运行完后面的区块的时候被求值。当该表达式的值为False时，在while后面的下一条命令将被执行。

Ren'Py没有continue，break或者for命令。Continue和break命令可以用jump命令替代。上面第一个while的例子展示了如何使用while来取代for命令。

值得一提的是，以上所有的True和False并不只是指这两个值，而是按照Python的规则来的。也就是说，所有不是0,False,[],{},""的对象都是True，而这些是False。用简单的话来说，如果一个变量的值“有点什么”，那它就是True，否则就是False（什么也没有）。

Pass命令¶

pass命令是当你需要一个区块但是什么事都不想做的时候使用的命令。它只是个占位符，什么也不会发生。

例如:

if points >= 10:
    "干得不错！"
elif points >= 1:
    pass
else:
    "情况有些不妙啊。"

# 当还有事件没被执行时，event.step()
# 会返回True。

while event.step():
    pass
